## Tasks from Rosetta Code implemented in Ballerina

### Build

```bash
# Clone the repo
git clone https://github.com/salif/ballerina-rosettacode.git
# Build
bal build
# Run
java -jar target/bin/rosettacode.jar -- "Hello_world/Text"
```
